# coCO4

## Description

coCO4 est un projet orienté analyse, expérimentation et optimisation de stratégies, notamment autour des données sportives et des prédictions.

L’objectif principal est de construire des outils simples et efficaces pour tester différentes approches (comme greedy, random, balanced, etc.) et identifier celles qui sont les plus performantes et les plus stables.

## Fonctionnalités

- Analyse de données de matchs
- Simulation de stratégies
- Comparaison de performances
- Optimisation de choix combinatoires
- Possibilité d’ajouter de nouvelles méthodes

## Objectifs

- Tester des stratégies de manière structurée
- Trouver des approches fiables et reproductibles
- Améliorer progressivement les performances
- Garder un code simple, clair et évolutif

## Technologies

- JavaScript
- HTML / CSS (si interface)
